# Arrase Tmux Theme

A tmux theme forked from [catppuccin-tmux](https://github.com/catppuccin/tmux) with native [Matugen](https://github.com/InioX/matugen) support for automatic color generation from your wallpaper.

![License](https://img.shields.io/github/license/arrase21/tmux-matugen)

## Features

- 🎨 **Matugen integration** — colors update automatically when you change your wallpaper
- 🔄 **Hotreload** — no manual tmux restart needed
- 📦 **Modular statusline** — mix and match status modules
- 🎛️ **Fallback theme** — works without Matugen via the built-in `arrase.tmuxtheme`

## Requirements

- tmux 3.0+
- [TPM](https://github.com/tmux-plugins/tpm) (optional)
- [Matugen](https://github.com/InioX/matugen) (optional, for dynamic colors)
- A [Nerd Font](https://www.nerdfonts.com/) for icons

## Installation

### With TPM

Add to your `~/.config/tmux/tmux.conf`:

```bash
set -g @plugin 'arrase21/tmux-matugen'
run '~/.tmux/plugins/tpm/tpm'
```

Then press `prefix + I` to install.

### Manual

```bash
git clone https://github.com/arrase21/tmux-matugen ~/.config/tmux/plugins/arrase
```

Add to your `~/.config/tmux/tmux.conf`:

```bash
run '~/.config/tmux/plugins/arrase/arrase.tmux'
```

## Matugen Setup

1. Add a template to your Matugen config (`~/.config/matugen/config.toml`):

```toml
[templates.tmux]
input_path = '~/.config/matugen/templates/tmux-colors.conf'
output_path = '~/.config/tmux/generated.conf'
post_hook = 'tmux source-file ~/.config/tmux/generated.conf && tmux source-file ~/.config/tmux/tmux.conf'
```

2. Create the template at `~/.config/matugen/templates/tmux-colors.conf`:

```bash
# Auto-generated by Matugen
set -g  status-bg                 "{{colors.surface_container_lowest.default.hex}}"
set -gq @thm_bar_bg               "{{colors.surface_container_lowest.default.hex}}"
set -gq @thm_bg                   "{{colors.surface.default.hex}}"
set -gq @thm_fg                   "{{colors.on_surface.default.hex}}"
set -gq @thm_primary              "{{colors.primary.default.hex}}"
set -gq @thm_inverse_primary      "{{colors.inverse_primary.default.hex}}"
set -gq @thm_surface_low          "{{colors.surface_container_low.default.hex}}"
set -gq @thm_surface              "{{colors.surface_container.default.hex}}"
set -gq @thm_surface_variant      "{{colors.surface_container_high.default.hex}}"
set -gq @thm_outline              "{{colors.outline_variant.default.hex}}"
set -gq @thm_text_variant         "{{colors.on_surface_variant.default.hex}}"
set -gq @thm_green                "{{colors.tertiary.default.hex}}"
set -gq @thm_red                  "{{colors.error.default.hex}}"
set -g  status-style              "bg=#{@thm_bg},fg=#{@thm_fg}"
set -g  window-active-style       "bg=#{@thm_bg},fg=#{@thm_fg}"
```

3. Source the generated file before the plugin in your `tmux.conf`:

```bash
source-file ~/.config/tmux/generated.conf
run '~/.config/tmux/plugins/arrase/arrase.tmux'
```

4. Run Matugen — tmux updates automatically:

```bash
matugen image /path/to/your/wallpaper.jpg
```

## Configuration

### Window Options

```bash
set -g @window_left_separator ""
set -g @window_right_separator " "
set -g @window_middle_separator " █"
set -g @window_number_position "right"  # left, right
set -g @window_default_fill "number"    # number, all, none
set -g @window_default_text "#W"
set -g @window_current_fill "number"
set -g @window_current_text "#W#{?window_zoomed_flag,(),}"
```

### Status Options

```bash
set -g @status_modules_right "directory session"
set -g @status_modules_left ""
set -g @status_left_separator  " "
set -g @status_right_separator " "
set -g @status_right_separator_inverse "no"
set -g @status_fill "icon"        # icon, all
set -g @status_connect_separator "no"
```

### Available Modules

| Module | Description |
|---|---|
| `session` | Current tmux session name |
| `directory` | Current working directory |
| `application` | Active pane application |
| `user` | Current user |
| `host` | Hostname |
| `date_time` | Date and time |
| `battery` | Battery status |

### Session Module Colors

The session module changes color when the tmux prefix is active:

```bash
set -g @session_color "#{?client_prefix,$thm_red,$thm_green}"
set -g @session_icon ""
set -g @session_text "#S"
```
